lightning-input-namemodule

Represents a name compound field.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
disabledglobalIf present, the input name field is disabled and users cannot interact with it.
field-level-helpglobal
fields-to-displayglobal[{type=string, value=firstName}, {type=string, value=salutation}, {type=string, value=lastName}]List of fields to be displayed on the component. This value defaults to ['firstName', 'salutation', 'lastName']. Other field values include middleName, informalName, suffix.
first-nameglobalDisplays the First Name field.
first-name-labelglobalReserved for internal use.
informal-nameglobalDisplays the Informal Name field.
informal-name-labelglobalReserved for internal use.
labelglobalThe label of the input name field.
last-nameglobalDisplays the Last Name field.
last-name-labelglobalReserved for internal use.
middle-nameglobalDisplays the Middle Name field.
middle-name-labelglobalReserved for internal use.
optionsglobalDisplays a list of salutation options, such as Dr. or Mrs., provided as label-value pairs.
read-onlyglobalIf present, the input name field is read-only and cannot be edited.
requiredglobalIf present, the input name field must be filled out before the form is submitted. A red asterisk is displayed on the Last Name field. An error message is displayed if a user interacts with the Last Name field and does not provide a value.
salutationglobalDisplays the Salutation field as a dropdown menu. Use the options attribute to provide salutations in an array of label-value pairs.
salutation-labelglobalReserved for internal use.
suffixglobalDisplays the Suffix field.
suffix-labelglobalReserved for internal use.
validityglobalRepresents the validity states that an element can be in, with respect to constraint validation.
variantglobalThe variant changes the appearance of a name compound field. Accepted variants include standard, label-hidden, label-inline, and label-stacked. This value defaults to standard. Use label-hidden to hide the label but make it available to assistive technology. Use label-inline to horizontally align the label and name fields. Use label-stacked to place the label above the name fields.
