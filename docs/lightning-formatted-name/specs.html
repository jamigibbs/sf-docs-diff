lightning-formatted-namemodule

Displays a formatted name that can include a salutation and suffix.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
first-nameglobalThe value for the first name.
formatgloballongThe format to use to display the name. Valid values include short, medium, and long. This value defaults to long.
informal-nameglobalThe value for the informal name.
last-nameglobalThe value for the last name.
middle-nameglobalThe value for the middle name.
salutationglobalThe value for the salutation, such as Dr. or Mrs.
suffixglobalThe value for the suffix, such as Jr. or Esq.
