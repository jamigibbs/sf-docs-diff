lightning-slidermodule

An input range slider for specifying a value between two specified numbers.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
disabledglobalIf present, the slider is disabled and users cannot interact with it.
labelglobalYesText label to describe the slider. Provide your own label to describe the slider.
maxglobalThe maximum value of the input range. The default is 100.
message-when-bad-inputglobalError message to be displayed when a bad input is detected.
message-when-pattern-mismatchglobalError message to be displayed when a pattern mismatch is detected.
message-when-range-overflowglobalError message to be displayed when a range overflow is detected.
message-when-range-underflowglobalError message to be displayed when a range underflow is detected.
message-when-step-mismatchglobalError message to be displayed when a step mismatch is detected.
message-when-too-longglobalError message to be displayed when the value is too long.
message-when-type-mismatchglobalError message to be displayed when a type mismatch is detected.
message-when-value-missingglobalError message to be displayed when the value is missing.
minglobalThe minimum value of the input range. The default is 0.
sizeglobalThe size of the slider. The default is an empty string, which sets the slider to the width of the viewport. Accepted values are x-small, small, medium, and large.
stepglobalThe step increment value of the input range. Example steps include 0.1, 1, or 10. The default is 1.
typeglobalhorizontalThe type determines the orientation of the slider. Accepted values are vertical and horizontal. The default is horizontal.
validityglobalRepresents the validity states of the slider input, with respect to constraint validation.
valueglobalThe numerical value of the slider. The default is 0.
variantglobalThe variant changes the appearance of the slider. Accepted variants include standard and label-hidden. The default is standard.

Methods

NameArgumentsAccessDescription
blurglobalRemoves keyboard focus from the input element.
checkValidityglobalReturns the valid attribute value (Boolean) on the ValidityState object.
focusglobalSets focus on the input element.
reportValidityglobalDisplays the error messages and returns false if the input is invalid. If the input is valid, reportValidity() clears displayed error messages and returns true.
setCustomValidity
NameTypeDescription
messageunknownThe string that describes the error. If message is an empty string, the error message is reset.
globalSets a custom error message to be displayed when the slider value is submitted.
showHelpMessageIfInvalidglobalDisplays error messages on invalid fields. An invalid field fails at least one constraint validation and returns false when checkValidity() is called.
