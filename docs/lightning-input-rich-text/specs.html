lightning-input-rich-textmodule

A WYSIWYG editor with a customizable toolbar for entering rich text

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
custom-buttonsglobalReserved for internal use. Custom buttons to add to the toolbar.
disabledglobalIf present, the editor is disabled and users cannot interact with it.
disabled-categoriesglobalA comma-separated list of button categories to remove from the toolbar.
formatsglobalA list of allowed formats. By default, the list is computed based on enabled categories. The 'table' format is always enabled to support copying and pasting of tables if formats are not provided.
labelglobalThe label of the rich text editor.
label-visibleglobalfalseIf present, the label on the rich text editor is visible.
message-when-bad-inputglobalError message to be displayed when invalid input is detected.
placeholderglobalText that is displayed when the field is empty, to prompt the user for a valid entry.
share-with-entity-idglobalEntity ID to share the image with.
validglobalSpecifies whether the editor content is valid. If invalid, the slds-has-error class is added. This value defaults to true.
valueglobalThe HTML content in the rich text editor.
variantglobalThe variant changes the appearance of the toolbar. Accepted variant is bottom-toolbar which causes the toolbar to be displayed below the text box.
