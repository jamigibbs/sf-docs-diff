lightning-formatted-addressmodule

Displays a formatted address with a link to the given location on Google Maps. The link is opened in a new tab. A static map can be displayed with the address for better context.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
cityglobalThe city detail for the address.
countryglobalThe country detail for the address.
disabledglobalIf present, the address is displayed as plain text and cannot be clicked or focused on.
latitudeglobalThe latitude of the location if known. Latitude values must be within -90 and 90.
localeglobal@salesforce/i18n/localeThe locale of the address. The default value is 'en-US'.
longitudeglobalThe longitude of the location if known. Longitude values must be within -180 and 180.
postal-codeglobalThe postal code detail for the address.
provinceglobalThe province detail for the address.
show-static-mapglobalDisplays a static map of the location using Google Maps. This value defaults to false.
streetglobalThe street detail for the address.

Methods

NameArgumentsAccessDescription
clickglobalSimulates a mouse click on the address and navigates to Google Maps on a new tab.
