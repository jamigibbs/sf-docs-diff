lightning-output-fieldmodule

Represents a read-only display of a label, help text, and value for a field on a Salesforce object.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
field-classglobalA CSS class for the outer element, in addition to the component's base classes.
field-nameglobalThe API name of the field to be displayed.
variantglobalChanges the appearance of the output. Accepted variants include standard and label-hidden. This value defaults to standard.
