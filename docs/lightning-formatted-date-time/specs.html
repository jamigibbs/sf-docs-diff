lightning-formatted-date-timemodule

Displays formatted date and time.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
dayglobalAllowed values are numeric or 2-digit.
eraglobalAllowed values are narrow, short, or long.
hourglobalAllowed values are numeric or 2-digit.
hour12globalDetermines whether time is displayed as 12-hour. If false, time displays as 24-hour. The default setting is determined by the user's locale.
minuteglobalAllowed values are numeric or 2-digit.
monthglobalAllowed values are 2-digit, narrow, short, or long.
secondglobalAllowed values are numeric or 2-digit.
time-zoneglobalThe time zone for date and time display. Use this attribute only if you want to override the default, which is the time zone set on the user device. Specify a time zone from the IANA time zone database (https://www.iana.org/time-zones). For example, set the value to 'Pacific/Honolulu' to display Hawaii time. The short code UTC is also accepted.
time-zone-nameglobalAllowed values are short or long. For example, the Pacific time zone would display as 'PST' if you specify 'short', or 'Pacific Standard Time' if you specify 'long.'
valueglobalThe value to be formatted, which can be a Date object, timestamp, or an ISO8601 formatted string.
weekdayglobalSpecifies how to display the day of the week. Allowed values are narrow, short, or long.
yearglobalAllowed values are numeric or 2-digit.
