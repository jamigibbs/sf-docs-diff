lightning-formatted-urlmodule

Displays a URL as a hyperlink.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
labelglobalThe text to display in the link.
tab-indexglobalReserved for internal use. Use tabindex instead to indicate if an element should be focusable. A value of 0 means that the element is focusable and participates in sequential keyboard navigation. A value of -1 means that the element is focusable but does not participate in keyboard navigation.
targetglobalSpecifies where to open the link. Options include _blank, _parent, _self, and _top. This value defaults to _self.
tooltipglobalThe text to display when the mouse hovers over the link. A link doesn't display a tooltip unless a text value is provided.
valueglobalThe URL to format.

Methods

NameArgumentsAccessDescription
blurglobalRemoves keyboard focus from the element.
clickglobalSimulates a mouse click on the url and navigates to it using the specified target.
focusglobalSets focus on the element.
