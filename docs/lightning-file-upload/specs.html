lightning-file-uploadmodule

A file uploader for uploading and attaching files to records.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
acceptglobalComma-separated list of file extensions that can be uploaded in the format .ext, such as .pdf, .jpg, or .png.
disabledglobalSpecifies whether this component should be displayed in a disabled state. Disabled components can't be clicked. The default is false.
file-field-nameglobalName of a custom field on the ContentVersion object. Set its value with the file-field-value attribute.
file-field-valueglobalValue to store in the custom field specified by file-field-name for the uploaded file.
labelglobalYesThe text label for the file uploader.
multipleglobalSpecifies whether a user can upload more than one file simultaneously. The default is false.
nameglobalYesSpecifies the name of the input element.
record-idglobalThe record Id of the record that the uploaded file is associated to.
