lightning-checkbox-groupmodule

A checkbox group that enables selection of single or multiple options.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
disabledglobalIf present, the checkbox group is disabled. Checkbox selections can't be changed for a disabled checkbox group.
labelglobalYesText label for the checkbox group.
message-when-value-missingglobalOptional message to be displayed when no checkbox is selected and the required attribute is set.
nameglobalYesThe name of the checkbox group.
optionsglobalYesArray of label-value pairs for each checkbox.
requiredglobalIf present, at least one checkbox must be selected.
validityglobalRepresents the validity states that an element can be in, with respect to constraint validation. Returns the ValidityState object for the checkbox group.
valueglobalYesThe list of selected checkboxes. Each array entry contains the value of a selected checkbox. The value of each checkbox is set in the options attribute.
variantglobalThe variant changes the appearance of the checkbox group. Accepted variants include standard, label-hidden, label-inline, and label-stacked. This value defaults to standard. Use label-hidden to hide the label but make it available to assistive technology. Use label-inline to horizontally align the label and checkbox group. Use label-stacked to place the label above the checkbox group.

Methods

NameArgumentsAccessDescription
checkValidityglobalReturns the valid attribute value (Boolean) on the ValidityState object.
focusglobalSets focus on the first checkbox input element.
reportValidityglobalDisplays the error messages and returns false if the input is invalid. If the input is valid, reportValidity() clears displayed error messages and returns true.
setCustomValidity
NameTypeDescription
messageunknownThe string that describes the error. If message is an empty string, the error message is reset.
globalSets a custom error message to be displayed when the checkbox value is submitted.
showHelpMessageIfInvalidglobalDisplays an error message if the checkbox value is required and no option is selected.
