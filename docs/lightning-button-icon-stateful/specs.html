lightning-button-icon-statefulmodule

An icon-only button that retains state.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
alternative-textglobalThe alternative text used to describe the icon. This text should describe what happens when you click the button, for example 'Upload File', not what the icon looks like, 'Paperclip'.
icon-nameglobalThe Lightning Design System name of the icon. Names are written in the format 'utility:down' where 'utility' is the category, and 'down' is the specific icon to be displayed. Only utility icons can be used in this component.
nameglobalThe name for the button element. This value is optional and can be used to identify the button in a callback.
selectedglobalSpecifies whether the button is in a selected state. This value defaults to false.
sizeglobalmediumThe size of the button-icon component. Options include xx-small, x-small, small, and medium. This value defaults to medium.
valueglobalThe value for the button element. This value is optional and can be used when submitting a form.
variantglobalborderThe variant changes the appearance of button-icon. Accepted variants include border, border-filled, and border-inverse. This value defaults to border.

Methods

NameArgumentsAccessDescription
focusglobalSets focus on the button.
