lightning-input-addressmodule

Represents an address compound field.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
address-labelglobalThe label for the address compound field.
address-lookup-placeholderglobalThe placeholder for the address lookup field option. Visible only when using show-address-lookup.
cityglobalThe value for the city field. Maximum length is 40 characters.
city-labelglobalThe label for the city field.
city-placeholderglobalThe placeholder for the city field.
countryglobalThe country field for the address. If country-options is provided, this country value is selected by default. Maximum length is 80 characters.
country-labelglobalThe label for the country field.
country-optionsglobalThe array of label-value pairs for the country. Displays a dropdown menu of options.
country-placeholderglobalThe placeholder for the country field.
disabledglobalIf present, the address fields are disabled and users cannot interact with them.
field-level-helpglobalHelp text detailing the purpose and function of the input.
postal-codeglobalThe value for postal code field. Maximum length is 20 characters.
postal-code-labelglobalThe label for the postal code field.
postal-code-placeholderglobalThe placeholder for the postal code field.
provinceglobalThe province field for the address. If province-options is provided, this province value is selected by default. Maximum length is 80 characters.
province-labelglobalThe label for the province field.
province-optionsglobalThe array of label-value pairs for the province. Displays a dropdown menu of options.
province-placeholderglobalThe placeholder for the province field.
read-onlyglobalIf present, the address fields are read-only and cannot be edited.
requiredglobalIf present, the address fields must be filled before the form is submitted.
show-address-lookupglobalIf present, address lookup using Google Maps is enabled.
streetglobalThe value for the street field. Maximum length is 255 characters.
street-labelglobalThe label for the street field.
street-placeholderglobalThe placeholder for the street field.
validityglobalRepresents the validity states that an element can be in, with respect to constraint validation.
variantglobalThe variant changes the appearance of an input address field. Accepted variants include standard, label-hidden, label-inline, and label-stacked. This value defaults to standard. Use label-hidden to hide the compound field label but make it available to assistive technology. Use label-inline to horizontally align the label and input address field. Use label-stacked to place the label above the input address field.

Methods
