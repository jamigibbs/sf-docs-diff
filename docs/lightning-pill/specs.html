lightning-pillmodule

A pill displays a label that can contain links and can be removed from view.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
aria-selectedglobalSpecifies the aria-selected of an element.
has-errorglobalIf present, the pill is shown with a red border and an error icon on the left of the label.
hrefglobalThe URL of the page that the link goes to.
is-plain-linkglobalReserved for internal use. Specifies whether the element variant is a plain link.
labelglobalYesThe text label that displays in the pill.
nameglobalThe name for the pill. This value is optional and can be used to identify the pill in a callback.
roleglobalSpecifies the role of an element.
tab-indexglobalReserved for internal use. Use tabindex instead to indicate if an element should be focusable. A value of 0 means that the pill is focusable and participates in sequential keyboard navigation. A value of -1 means that the pill is focusable but does not participate in keyboard navigation.
variantglobalThe variant changes the appearance of the pill. Accepted variants include link, plain, and plainLink. The default variant is link, which creates a link in the pill when you specify the href attribute. The plain variant renders the pill without a link and ignores the href attribute. The plainLink variant is reserved for internal use.

Methods
