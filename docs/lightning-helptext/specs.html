lightning-helptextmodule

An icon with a text popover used for tooltips.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
alternative-textglobalThe assistive text for the button icon. The default is "Help". Screen readers announce the assistive text and help text content as {alternativeText} button {content}. If not set, screen readers announce "Help button {content}". The text should describe the function of the icon, for example, "Show help text".
contentglobalText to be shown in the popover.
icon-nameglobal
icon-variantglobal

Methods

NameArgumentsAccessDescription
focusglobalSets focus on the button.
