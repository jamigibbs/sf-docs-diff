lightning-dynamic-iconmodule

Represents various animated icons with different states.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
alternative-textglobalThe alternative text used to describe the dynamic icon. This text should describe what's happening. For example, 'Graph is refreshing', not what the icon looks like, 'Graph'.
optionglobalThe option attribute changes the appearance of the dynamic icon. The options available depend on the type. eq: play (default) or stop score: positive (default) or negative strength: -3, -2, -1, 0 (default), 1, 2, 3 trend: neutral (default), up, or down
typeglobalYesThe Lightning Design System name of the dynamic icon. Valid values are: ellie, eq, score, strength, trend, and waffle.
