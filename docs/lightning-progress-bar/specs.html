lightning-progress-barmodule

Displays a horizontal progress bar from left to right to indicate the progress of an operation.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
aria-labelglobalDescribes the input to assistive technologies.
sizeglobalThe size of the progress bar. Valid values are x-small, small, medium, and large. The default value is medium.
valueglobal0The percentage value of the progress bar.
variantglobalThe variant changes the appearance of the progress bar. Accepted variants include base or circular. This value defaults to base.
