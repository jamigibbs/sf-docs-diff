lightning-buttonmodule

A clickable element used to perform an action.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
disable-animationglobalReserved for internal use. If present, disables button animation.
icon-nameglobalThe Lightning Design System name of the icon. Names are written in the format 'utility:down' where 'utility' is the category, and 'down' is the specific icon to be displayed.
icon-positiongloballeftDescribes the position of the icon with respect to the button label. Options include left and right. This value defaults to left.
labelglobalThe text to be displayed inside the button.
nameglobalThe name for the button element. This value is optional and can be used to identify the button in a callback.
typeglobalbuttonSpecifies the type of button. Valid values are button, reset, and submit. This value defaults to button.
valueglobalThe value for the button element. This value is optional and can be used when submitting a form.
variantglobalneutralThe variant changes the appearance of the button. Accepted variants include base, neutral, brand, brand-outline, destructive, destructive-text, inverse, and success. This value defaults to neutral.

Methods

NameArgumentsAccessDescription
clickglobalSimulates a mouse click on the button.
focusglobalSets focus on the button.
