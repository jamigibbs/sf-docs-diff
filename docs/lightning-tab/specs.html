lightning-tabmodule

A single tab in a tabset component.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
end-icon-alternative-textglobalThe alternative text for the icon specified by end-icon-name.
end-icon-nameglobalThe Lightning Design System name of an icon to display at the end of the tab label. Specify the name in the format 'utility:check' where 'utility' is the category, and 'check' is the icon to be displayed.
icon-assistive-textglobalThe alternative text for the icon specified by icon-name.
icon-nameglobalThe Lightning Design System name of an icon to display at the beginning of the tab label. Specify the name in the format 'utility:down' where 'utility' is the category, and 'down' is the icon to be displayed. Only utility icons can be used.
labelglobalThe text displayed in the tab header.
show-error-indicatorglobalSpecifies whether there's an error in the tab content. An error icon is displayed to the right of the tab label.
titleglobalSpecifies text that displays in a tooltip over the tab content.
valueglobalThe optional string to identify which tab was clicked during the tab's active event. This string is also used by active-tab-value in tabset to open a tab.
