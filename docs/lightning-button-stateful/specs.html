lightning-button-statefulmodule

A button that toggles between states.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
group-orderglobalReserved for internal use only. Describes the order of this element (first, middle or last) inside lightning-button-group.
icon-name-when-hoverglobalThe name of the icon to be used in the format 'utility:close' when the state is true and the button receives focus.
icon-name-when-offglobalThe name of the icon to be used in the format 'utility:add' when the state is false.
icon-name-when-onglobalThe name of the icon to be used in the format 'utility:check' when the state is true.
label-when-hoverglobalThe text to be displayed inside the button when state is true and the button receives focus.
label-when-offglobalYesThe text to be displayed inside the button when state is false.
label-when-onglobalYesThe text to be displayed inside the button when state is true.
selectedglobalIf present, the button is in the selected state.
variantglobalneutralThe variant changes the appearance of the button. Accepted variants include brand, destructive, inverse, neutral, success, and text.

Methods

NameArgumentsAccessDescription
focusglobalSets focus on the button.
