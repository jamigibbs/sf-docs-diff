lightning-input-fieldmodule

Represents an editable input for a field on a Salesforce object.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
dirtyglobalReserved for internal use. If present, the field has been modified by the user but not saved or submitted.
disabledglobalIf present, the field is grayed out and users can't interact with it. Disabled fields don't receive focus and are skipped in tabbing navigation.
field-nameglobal
read-onlyglobalSpecifies whether an input field is read-only. This value defaults to false. Not supported for the following field types: rich text, picklist, multi-select picklist, and lookup. A read-only field is not disabled by default.
requiredglobalIf present, the input field must be filled out before the form is submitted.
valueglobalThe field value, which overrides the existing value.
variantglobalThe variant changes the label position of an input field. Accepted variants include standard, label-hidden, label-inline, and label-stacked. The variant, if specified, determines the label position. Otherwise, the density setting of the parent form determines the label position.
