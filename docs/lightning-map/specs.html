lightning-mapmodule

Displays a map with markers for one or more locations.

Support:BETA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
centerglobalA location to use as the map's center. If center is not specified, the map centers automatically.
list-viewglobalautoDisplays or hides the list of locations. Valid values are visible, hidden, or auto. This value defaults to auto, which shows the list only when multiple markers are present. Passing in an invalid value hides the list view.
map-markersglobalSetter function, for mapMarkers. Depending on the number of markers passed, we display a single view map or a map with multiple markers and a list of coordinates
markers-titleglobalSetter for the markersTitle property.
optionsglobalProvides the list of map settings/options. The options contain different map settings with either true/false value.
selected-marker-valueglobalProvides the value of the currently selected marker. Returns undefined if you donâ€™t pass value to map-markers.
show-footerglobalfalseIf present, the footer element is displayed below the map. The footer shows an 'Open in Google Maps' link that opens an external window to display the selected marker location in Google Maps. Default value is false.
zoom-levelglobalThe zoom levels as defined by Google Maps API. If a zoom level is not specified, a default zoom level is applied to accommodate all markers on the map.
