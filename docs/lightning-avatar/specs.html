lightning-avatarmodule

A visual representation of an object.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
alternative-textglobalYesThe alternative text used to describe the avatar, which is displayed as hover text on the image.
fallback-icon-nameglobalThe Lightning Design System name of the icon used as a fallback when the image fails to load. The initials fallback relies on this for its background color. Names are written in the format 'standard:account' where 'standard' is the category, and 'account' is the specific icon to be displayed. Only icons from the standard and custom categories are allowed.
initialsglobalIf the record name contains two words, like first and last name, use the first capitalized letter of each. For records that only have a single word name, use the first two letters of that word using one capital and one lower case letter.
sizeglobalThe size of the avatar. Valid values are x-small, small, medium, and large. This value defaults to medium.
srcglobalYesThe URL for the image.
variantglobalThe variant changes the shape of the avatar. Valid values are empty, circle, and square. This value defaults to square.
