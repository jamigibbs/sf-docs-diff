lightning-comboboxmodule

A widget that provides an input field that is readonly, accompanied by a dropdown list of selectable options.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
aria-described-byglobalReserved for internal use. Use the standard aria-describedby instead. A space-separated list of element IDs that provide descriptive labels for the combobox.
aria-labelled-byglobalReserved for internal use. Use the standard aria-labelledby instead. A space-separated list of element IDs that provide labels for the combobox.
autocompleteglobalReserved for internal use. Controls auto-filling of the field. Set the attribute to pass through autocomplete values to be interpreted by the browser.
disabledglobalIf present, the combobox is disabled and users cannot interact with it.
dropdown-alignmentgloballeftSpecifies where the drop-down list is aligned with or anchored to the selection field. By default the list is aligned with the selection field at the top left so the list opens down. Use bottom-left to make the selection field display at the bottom so the list opens above it. Use auto to let the component determine where to open the list based on space available.
field-level-helpglobalHelp text detailing the purpose and function of the combobox.
labelglobalText label for the combobox.
message-when-value-missingglobalError message to be displayed when the value is missing and input is required.
nameglobalSpecifies the name of the combobox.
optionsglobalYesA list of options that are available for selection. Each option has the following attributes: label and value.
placeholderglobalText that is displayed before an option is selected, to prompt the user to select an option. The default is "Select an Option".
read-onlyglobalIf present, the combobox is read-only. A read-only combobox is also disabled.
requiredglobalIf present, a value must be selected before the form can be submitted.
spinner-activeglobalIf present, a spinner is displayed below the menu items to indicate loading activity.
validityglobalYesRepresents the validity states that an element can be in, with respect to constraint validation.
valueglobalSpecifies the value of an input element.
variantglobalThe variant changes the appearance of the combobox. Accepted variants include standard, label-hidden, label-inline, and label-stacked. This value defaults to standard. Use label-hidden to hide the label but make it available to assistive technology. Use label-inline to horizontally align the label and combobox. Use label-stacked to place the label above the combobox.
