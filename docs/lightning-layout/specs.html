lightning-layoutmodule

Represents a responsive grid system for arranging containers on a page.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
horizontal-alignglobalDetermines how to spread the layout items horizontally. The alignment options are center, space, spread, and end.
multiple-rowsglobalIf present, layout items wrap to the following line when they exceed the layout width.
pull-to-boundaryglobalPulls layout items to the layout boundaries and corresponds to the padding size on the layout item. Possible values are small, medium, or large.
vertical-alignglobalDetermines how to align the layout items vertically in the container. The alignment options are start, center, end, and stretch.
