lightning-button-menumodule

Represents a dropdown menu with a list of actions or functions.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
access-keyglobalThe keyboard shortcut for the button menu.
alternative-textglobalThe assistive text for the button.
disabledglobalIf present, the menu can be opened by users.
draft-alternative-textglobalDescribes the reason for showing the draft indicator. This is required when is-draft is true.
icon-nameglobalutility:downThe name of the icon to be used in the format 'utility:down'. If an icon other than 'utility:down' or 'utility:chevrondown' is used, a utility:down icon is appended to the right of that icon. This value defaults to utility:down.
icon-sizeglobalmediumThe size of the icon. Options include xx-small, x-small, small, medium, or large. This value defaults to medium.
is-draftglobalIf present, the menu trigger shows a draft indicator.
is-loadingglobalIf present, the menu is in a loading state and shows a spinner.
labelglobalOptional text to be shown on the button.
loading-state-alternative-textglobalMessage displayed while the menu is in the loading state.
menu-alignmentglobalDetermines the alignment of the menu relative to the button. Available options are: auto, left, center, right, bottom-left, bottom-center, bottom-right. The auto option aligns the dropdown menu based on available space. This value defaults to left.
nubbinglobalIf present, a nubbin is present on the menu. A nubbin is a stub that protrudes from the menu item towards the button menu. The nubbin position is based on the menu-alignment.
titleglobalDisplays tooltip text when the mouse moves over the button menu.
tooltipglobalText to display when the user mouses over or focuses on the button. The tooltip is auto-positioned relative to the button and screen space.
valueglobalThe value for the button element. This value is optional and can be used when submitting a form.
variantglobalThe variant changes the look of the button. Accepted variants include bare, container, border, border-filled, bare-inverse, and border-inverse. This value defaults to border.

Methods

NameArgumentsAccessDescription
clickglobalSimulates a mouse click on the button.
focusglobalSets focus on the button.

Slots

NameDescription
defaultPlaceholder for menu-item
