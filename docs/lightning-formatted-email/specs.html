lightning-formatted-emailmodule

Displays an email as a hyperlink with the mailto: URL scheme.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
hide-iconglobalfalseIf present, the email icon is hidden and only the email address is displayed.
labelglobalThe text label for the email address.
tab-indexglobalReserved for internal use. Use tabindex instead to indicate if an element should be focusable. A value of 0 means that the element is focusable and participates in sequential keyboard navigation. A value of -1 means that the element is focusable but does not participate in keyboard navigation.
valueglobalYesThe email address that's displayed if a label is not provided.

Methods

NameArgumentsAccessDescription
blurglobalRemoves keyboard focus from the element.
clickglobalSimulates a mouse click on the email address and opens the default email app.
focusglobalSets focus on the element.
