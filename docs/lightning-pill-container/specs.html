lightning-pill-containermodule

A list of pills grouped in a container. This component requires API version 42.0 and later.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
is-collapsibleglobalSpecifies whether the pill container can be collapsed.
is-expandedglobalSpecifies whether the pill container is expanded.
itemsglobalAn array of items to be rendered as pills in a container.
labelglobalAria label for the pill container.
single-lineglobalSpecifies whether to keep the pills in a single line.
variantglobalThe variant changes the appearance of the pill container. Accepted variants include standard and bare. This value defaults to standard.

Methods

NameArgumentsAccessDescription
focusglobalSets focus on the pill list.
