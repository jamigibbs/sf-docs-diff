lightning-pill-containermodule

A list of pills grouped in a container. This component requires API version 42.0 and later.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
is-collapsibleglobalSpecifies whether the pill list can be collapsed. Use is-collapsible with the is-expanded attribute to expand and collapse the list of pills.
is-expandedglobalSpecifies whether the list of pills is expanded or collapsed, when is-collapsible is true. This attribute is ignored when is-collapsible is false, and the list of pills is expanded even if is-expanded is false or not set.
itemsglobalAn array of pill attribute values that define pills to display in the container.
labelglobalAria label for the pill container to describe the list of options.
single-lineglobalSpecifies whether to limit pill display to one line. This attribute overrides the is-collapsible and is-expanded attributes.
variantglobalThe variant changes the tab navigation behavior of the pill container. Accepted variants include standard and bare. This value defaults to standard which supports accessibility.

Methods

NameArgumentsAccessDescription
focusglobalSets focus on the pill list.
