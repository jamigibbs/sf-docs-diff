lightning-badgemodule

Represents a label which holds a small amount of information, such as the number of unread notifications.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
icon-alternative-textglobalThe alternative text used to describe the icon, which is displayed as tooltip text.
icon-nameglobalThe Lightning Design System name of the icon to be displayed inside the badge. Names are written in the format 'utility:down' where 'utility' is the category, and 'down' is the specific icon to be displayed.
icon-positionglobalThe position for the icon. Possible values: 'start' (displayed before the text) and 'end' (displayed after the text)
labelglobalYesThe text to be displayed inside the badge.
