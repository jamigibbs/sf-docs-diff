Support:GA
Access:global

NameTypeAccessRequiredDefaultDescription
alternative-textglobalThe alternative text used to describe the reason for the wait and need for a spinner.

variantglobalThe variant changes the appearance of the spinner. Accepted variants include base, brand, and inverse. The default is base.
